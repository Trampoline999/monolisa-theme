# Monolisa Theme

A beautiful, dark custom theme for Visual Studio Code, inspired by Motion Blue but with customized highlights and editor styling.

## Features

- Deep, rich blue background (`#183752`)
- Custom semantic highlighting
- Unique color palette tailored for readability

## Installation

1. Open the Extensions sidebar in VS Code
2. Search for "Monolisa"
3. Click Install
4. Go to Preferences: Color Theme and select "Monolisa"

## Enjoy!
